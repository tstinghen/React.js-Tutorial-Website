We are going to add a function to the top of our List class, called getInitialState, which initializes item and due, in our info object, to empty strings.





State is like props, in that it refers to attributes of an object. State, however, is what we want to use when getting input from the user, because when state changes, React knows to update that specific part of the DOM (and not to update other parts of the DOM).

So, to access an attribute's state we would say

{this.state.attributeName}





The form also needs some more information about the object's states.

We'll add information to each input field about the input's value, and onChange action.

This new data will associate the state of item and due with the text entered into the form, and also names two new functions that we will use to handle the data that is entered: changeInfo, and changeData.
