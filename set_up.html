This is the starting point for a React app. Like any .js library, you must include the library in your html file.

In addition to the React library, which can be linked from online, as shown, or downloaded from React , you may include the JSX Transformer, which converts JSX (a markup language similar to HTML) into Javascript.

React uses JSX to make code that is clear and readable, and to make it easy to include markup information directly in the React script. JSX is optional when using React, but it is recommended, and we will use it here. JSX syntax and help can be found at the React.js site.





Note the comment below the script tag. This comment is necessary in order for JSXTransform to work properly! Your JSX will not work without this comment!
